Network Security Lab

Designed and stress-tested VLAN, ACL, and port security scenarios, exploring edge cases in segmentation and access control in a controlled lab environment.

01. OVERVIEW

In a modern enterprise, a "flat" network is a playground for lateral movement. This lab focuses on Micro-segmentation and Access Control Lists (ACLs) to ensure that even if one segment is compromised (like a Guest Wi-Fi or IoT device), the core production data remains isolated and secure.

02. METHODOLOGY

  • VLAN Segmentation:Established logically isolated networks (Management, Production, IoT, and DMZ) to reduce the broadcast domain and attack surface.
  • Switch-Port Security: Configured MAC-address limiting and "Sticky" MACs to prevent rogue device injection and CAM table overflow attacks.
  • DHCP Snooping & Dynamic ARP Inspection (DAI): Mitigated Man-in-the-Middle (MitM) attacks by ensuring only authorized DHCP servers can assign IPs and validating ARP packets.
